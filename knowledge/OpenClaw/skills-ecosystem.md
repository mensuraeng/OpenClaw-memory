# skills-ecosystem.md

## Escopo
Camada de capacidades distribuídas, registro e descoberta de skills.

## Repositórios principais
- `openclaw/clawhub` — diretório de skills do OpenClaw.
- `openclaw/skills` — arquivo/versionamento das skills publicadas.

## Quando consultar este arquivo
- publicação de skills
- descoberta de skills
- catálogo e ecossistema de capacidades
- relação entre skill local, skill publicada e hub

## Leitura correta
A capacidade do ecossistema não está só no core. O valor operacional do OpenClaw cresce muito via skills. `clawhub` e `skills` são parte do mapa real do produto ampliado.

## Regra prática
Se a dúvida for “isso é do produto base ou vem do ecossistema de skills?”, consultar este arquivo.
