# clients-and-nodes.md

## Escopo
Clientes auxiliares, nodes e plataformas complementares.

## Repositórios principais
- `openclaw/openclaw-windows-node` — suíte complementar para Windows, incluindo tray app, biblioteca compartilhada, node e extensão para PowerToys Command Palette.
- `openclaw/clawgo` — node do Clawd em Go.
- `openclaw/casa` — projeto em Swift para expor a base/home ao Clawdbot.
- `openclaw/flawd-bot` — projeto auxiliar listado pela organização.

## Quando consultar este arquivo
- nodes complementares
- clientes auxiliares
- integração com Windows
- variações de implementação por plataforma

## Leitura correta
Essa camada não substitui o core, mas amplia alcance operacional e suporte a ambientes específicos.

## Regra prática
Se a dúvida for sobre node, cliente auxiliar, tray app ou plataforma complementar, olhar aqui.
