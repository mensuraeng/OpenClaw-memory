# advanced-ops.md

## Escopo
Ferramentas oficiais de operação avançada, automação e composição.

## Repositórios principais
- `openclaw/lobster` — workflow shell nativo do OpenClaw; macro engine local-first para compor skills, tools e automações.
- `openclaw/acpx` — cliente CLI headless para sessões stateful do Agent Client Protocol (ACP).
- `openclaw/multipass` — CLI para testar canais de mensagem contra o OpenClaw.
- `openclaw/caclawphony` — plataforma para transformar trabalho de projeto em execuções autônomas isoladas.

## Quando consultar este arquivo
- automação avançada
- composição de workflows
- ACP e sessões stateful
- teste de canais
- execução autônoma isolada

## Leitura correta
Esses repositórios importam quando a operação já saiu do uso básico e entrou em composição, automação robusta e runtime avançado.

## Regra prática
Se a dúvida for sobre fluxo operacional avançado e não sobre uso básico do produto, olhar aqui antes de improvisar arquitetura.
