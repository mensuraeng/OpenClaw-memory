# Workflow — docling

## 1. Identificar a entrada
Classificar o arquivo:
- PDF nativo
- PDF escaneado
- imagem
- DOCX
- PPTX
- página/documento a estruturar

## 2. Definir objetivo
Escolher um objetivo claro:
- extrair texto
- gerar markdown
- fazer OCR
- preparar para análise
- preparar para ingestão

## 3. Executar com ambiente isolado
Sempre usar os binários em:
- `docling-env/bin/`

## 4. Validar saída
Checar:
- legibilidade
- estrutura
- títulos
- tabelas relevantes
- perda crítica de conteúdo

## 5. Decidir próximo passo
- entregar arquivo convertido
- seguir para resumo
- seguir para análise técnica
- seguir para ingestão/indexação
