# Sales Enablement

Use for one-pagers, decks, objection handling, proposal support and sales scripts.

## Assets that matter

- One-page diagnostic offer.
- Executive deck: problem → cost → method → proof → next step.
- Objection handling sheet.
- Case-based proof library.
- Discovery call script.

## One-pager structure

1. Problem in buyer language.
2. Why current control fails.
3. MENSURA/MIA/PCS method.
4. 3 outcomes.
5. Proof or case signal.
6. Next step.

## Objection framework

For each objection:

- Exact objection.
- Real concern behind it.
- Response approach.
- Proof point.
- Follow-up question.

## Guardrail

Do not overclaim savings, guarantee deadlines, accept responsibility, approve costs, or imply technical/legal acceptance without explicit validation.
