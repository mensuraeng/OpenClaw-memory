# .trash — lixeira operacional

Arquivos que seriam removidos por automação devem ser movidos para cá quando possível, em vez de apagados permanentemente.

Padrão recomendado de nome:

`YYYY-MM-DD__origem__motivo__nome-original`

Retenção alvo: 30 dias.

Não colocar segredos aqui sem criptografia; se o arquivo tiver segredo, mover para contenção segura e registrar sem expor conteúdo.
