Política de Privacidade

Mensura Engenharia Ltda. — CNPJ: [CNPJ] — Última atualização: abril de 2025

1. Quem somos

A Mensura Engenharia Ltda. é responsável pelo tratamento dos dados descritos nesta política. Contato: aa.csc4d@icloud.com

2. Dados coletados via LinkedIn API

Ao autorizar nossos aplicativos via LinkedIn, podemos coletar: nome completo, e-mail profissional, cargo, empresa, URL do perfil, foto de perfil e, para administradores de páginas, dados de engajamento e métricas das páginas de empresa (MENSURA Engenharia, MIA Engenharia, PCS Engenharia).

3. Finalidade do tratamento

Os dados são usados exclusivamente para: automação de publicações nas páginas de empresa no LinkedIn, análise de métricas de desempenho e gestão centralizada das páginas institucionais via plataforma OpenClaw.

4. Base legal

O tratamento é realizado com base no consentimento expresso do usuário (Art. 7º, I da LGPD) e no legítimo interesse da empresa para fins de comunicação institucional.

5. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos dados pessoais com terceiros para fins comerciais. Os dados podem ser processados por provedores de infraestrutura (servidores, APIs) exclusivamente para viabilizar os serviços descritos.

6. Retenção de dados

Os dados de autenticação (tokens OAuth) são mantidos apenas enquanto o acesso autorizado estiver ativo. Dados de métricas são retidos por no máximo 90 dias para fins analíticos.

7. Seus direitos

Você pode, a qualquer momento: revogar o acesso pelo painel do LinkedIn em Aplicativos autorizados, solicitar acesso, correção ou exclusão dos seus dados, e registrar reclamações junto à ANPD.

8. Contato para exclusão de dados

Envie solicitações de exclusão para: aa.csc4d@icloud.com

9. Alterações nesta política

Podemos atualizar esta política periodicamente. A data de "última atualização" no topo indica quando foi revisada pela última vez.
