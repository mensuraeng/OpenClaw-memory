# Pessoal do Alê

_Atualizado em 2026-04-20_

## Objetivo
Memória documental pessoal do Alê para consultas futuras.

## O que deve entrar aqui
- documentos pessoais importantes
- fichas cadastrais pessoais
- comprovantes e documentos-base recorrentes
- referências pessoais duráveis que precisem de consulta futura

## Estrutura recomendada
- `documentos-index.md` → índice dos documentos-fonte e anexos
- `cadastro-base.md` → base pessoal reutilizável quando fizer sentido
- `identificacao.md` → dados recorrentes que precisem ser reaproveitados

## Regra operacional
Sempre que entrar documento pessoal importante do Alê, registrar aqui o resumo estruturado e apontar a fonte original para consulta futura.
