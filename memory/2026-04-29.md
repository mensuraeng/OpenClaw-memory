
## Mission Control — edição de crons na página `/cron`

- Alê pediu criar capacidade de editar crons diretamente na página de Cron Jobs do Mission Control.
- Implementado botão `Edit` nos cards de cron e modal de edição.
- Campos editáveis v1: nome, agentId, enabled/paused, tipo de agenda (`cron`, `every`, `at`), expressão cron, timezone, everyMs, data ISO, sessionTarget, descrição e mensagem/payload.
- API `/api/cron` PUT reforçada para aceitar update consolidado com validações básicas de ID, agentId, sessionTarget, cron expression, everyMs e data ISO.
- Guardrail: continua sendo edição interna do scheduler; ações externas permanecem condicionadas às regras de aprovação fora do painel.
- Validação: `npm run build` em `projects/mission-control` concluiu com sucesso; PM2 `mission-control` reiniciado; rota externa `https://mc.mensuraengenharia.com.br/cron` responde `HTTP 307` para login com anti-cache.
