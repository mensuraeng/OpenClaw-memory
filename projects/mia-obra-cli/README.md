# mia-obra CLI

CLI read-only/dry-run para MIA obra, começando por CCSP Casa 7.

```bash
./bin/mia-obra status
./bin/mia-obra ccsp classify --text "alinhamento operacional"
./bin/mia-obra ccsp weekly-preview
./bin/mia-obra ccsp manha-dry-run
./bin/mia-obra ccsp rdo-dry-run
./bin/mia-obra guardrails
```

Sem envio externo no v0.1.
