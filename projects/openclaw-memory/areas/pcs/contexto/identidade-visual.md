# PCS — Identidade Visual

_Atualizado em 2026-04-13_

## Decisão adotada

Adotar como padrão visual oficial da PCS:
- o **logo horizontal com assinatura**
- a **paleta de cores oficial** enviada pelo Alê

## Diretriz de uso

### Papel das cores
- **Cinza escuro** → base institucional, sobriedade, robustez, estrutura
- **Warm red** → destaque, energia, ponto focal e assinatura de marca
- **Off white** → respiro, fundo claro, limpeza visual

### Leitura de marca
A identidade visual da PCS deve transmitir:
- robustez operacional
- clareza institucional
- autoridade técnica
- sobriedade
- capacidade real de execução

Evitar:
- estética premium/luxuosa da MIA
- visual excessivamente frio ou analítico como linguagem de PMO puro da MENSURA
- excesso de cores ou variações fora da paleta principal
