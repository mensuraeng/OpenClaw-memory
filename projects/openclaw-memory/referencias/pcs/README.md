# PCS Engenharia — Base de Conhecimento

_Atualizado em 2026-04-07_

## Objetivo

Esta pasta reúne referências técnicas e regulatórias usadas como base de conhecimento para temas ligados a calçadas, acessibilidade, execução urbana e conformidade técnica no município de São Paulo.

## Arquivos

### 1. `livreto-calcadas-uso-digital.pdf`
**Título provável:** Calçadas e Vias Exclusivas de Pedestres – Características Geométricas e Métodos Construtivos  
**Origem:** Prefeitura de São Paulo / SMPED / CPA  
**Utilidade para a PCS:** referência técnica principal para projeto e execução.

**Pontos-chave:**
- parâmetros geométricos de acessibilidade
- divisão da calçada em faixas
- inclinações máximas e condições de circulação
- tipos de rebaixamento de guia
- aplicação de piso tátil
- método construtivo em concreto moldado in loco

### 2. `cartilha-calcadas-draft-10.pdf`
**Título provável:** Conheça as Regras para Arrumar a Sua Calçada  
**Origem:** Prefeitura de São Paulo / Programa Passeio Livre  
**Utilidade para a PCS:** referência regulatória e didática para entendimento das regras urbanas e comunicação com cliente, fiscalização e subprefeitura.

**Pontos-chave:**
- organização das calçadas em 2 ou 3 faixas
- regras para acesso de veículos e esquinas
- critérios para vegetação e faixa livre
- materiais permitidos
- compilado legal sobre calçadas
- responsabilidades do proprietário e do poder público

### 3. `cartilha-de-calcadas-2020.pdf`
**Status:** versão duplicada ou equivalente do conteúdo técnico do livreto principal, com outra diagramação.  
**Utilidade para a PCS:** baixa utilidade adicional. Manter apenas como redundância documental.

## Leitura estratégica

A utilidade real dessa base para a PCS não é institucional, é operacional:
- apoiar propostas e memoriais técnicos
- embasar diretrizes de execução e fiscalização
- reduzir erro em detalhamento de acessibilidade
- sustentar conversas com cliente e órgão público com base normativa

## Recomendação de uso

Quando o tema envolver calçadas públicas, acessibilidade, reforma urbana ou adequação em São Paulo:
- usar o `livreto-calcadas-uso-digital.pdf` como referência técnica principal
- usar a `cartilha-calcadas-draft-10.pdf` como apoio regulatório e explicativo
- tratar `cartilha-de-calcadas-2020.pdf` como backup documental
