**PMO Torre de Controle** — três pilares:

1. **PMO Torre de Controle** — visão consolidada do portfólio, ritos de decisão, disciplina de execução
2. **Metodologia estruturada** — Last Planner System (LPS), Linha de Balanço, EVM, BIM 4D/5D
3. **Ritos e indicadores** — War Rooms quinzenais, dashboards executivos, métricas auditáveis

**4 eixos de monitoramento:**

| Eixo | Indicadores principais |
|---|---|
| Prazo | PPC (confiabilidade semanal), Baseline (desvio vs contratual), Caminho crítico |
| CAPEX | EAC (estimativa final), ETC (custo para terminar), Curva S (físico vs financeiro) |
| Produção | Avanço físico real, Produtividade RUP, Restrições de fluxo |
| Riscos | Top 5 ativos, Planos de mitigação, Não conformidades (NCR) |
