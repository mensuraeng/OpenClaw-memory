# Uso do academic-research-hub no stack do Alê

## Papel
Camada auxiliar de pesquisa aplicada para buscar papers e referências técnicas quando web search comum não for suficiente.

## Onde usar
- Lean Construction
- EVM
- BIM
- produtividade
- gestão de riscos
- governança técnica
- comparações metodológicas

## Onde gera valor
### MENSURA
- aprofundar repertório técnico
- embasar tese de método
- fortalecer propostas, materiais e argumentos

### MIA
- traduzir base técnica em proposta de valor premium
- sustentar diferenciação com substância

### PCS
- resolver problema técnico específico com base melhor

## O que evitar
- usar para juntar paper sem aplicação
- transformar pesquisa em fim em si mesma
- trazer densidade técnica que não vira decisão ou material útil

## Fluxo recomendado
1. formular pergunta técnica clara
2. buscar fontes acadêmicas
3. extrair 3 a 5 insights aplicáveis
4. converter em implicação prática para obra, proposta, processo ou posicionamento

## Regra prática
Se a pesquisa não virar ação, argumento, método ou diferenciação, não vale o custo.
