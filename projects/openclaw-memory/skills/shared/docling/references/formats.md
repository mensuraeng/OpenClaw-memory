# Formatos e casos de uso — docling

## Entradas típicas
- PDF técnico
- edital
- proposta
- memorial
- apresentação
- imagem escaneada
- documento fotografado

## Saídas desejadas
- texto limpo
- markdown estruturado
- conteúdo pronto para análise
- base para RAG / busca / indexação

## Quando preferir Docling
- documento complexo
- extração ruim com métodos simples
- necessidade de OCR
- necessidade de estrutura melhor que texto bruto

## Quando talvez não precise Docling
- PDF pequeno e legível já atendido pelo tool `pdf`
- texto simples que pode ser lido diretamente
