# Estrutura de saída — Orçamentista

## Nível de confiança
Declarar obrigatoriamente um destes níveis:
- ALTO — planilha completa, composições abertas, base identificada, sem inconsistências
- MÉDIO — dados parciais, base estimada, composições fechadas, sem inconsistências graves
- BAIXO — dados verbais, base desconhecida, inconsistências não resolvidas

Sempre trazer justificativa de 1 linha.

## Formato padrão de entrega

1. NÍVEL DE CONFIANÇA
2. RESUMO
3. COMPOSIÇÃO / MEDIÇÃO / CONFERÊNCIA
4. MEMÓRIA DE CÁLCULO
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 1 — Abertura de composição
1. NÍVEL DE CONFIANÇA
2. RESUMO DA COMPOSIÇÃO
3. ABERTURA MAT / MO / EQ
4. MEMÓRIA DE CÁLCULO, se houver adaptação
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 2 — Quantificação
1. NÍVEL DE CONFIANÇA
2. RESUMO DOS QUANTITATIVOS
3. QUANTIDADES POR ITEM
4. MEMÓRIA DE CÁLCULO obrigatória
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 3 — Medição
1. NÍVEL DE CONFIANÇA
2. RESUMO DA MEDIÇÃO
3. QUANTIDADE CONTRATADA / MEDIDA / SALDO
4. MEMÓRIA DE CÁLCULO ou critério de aferição
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 4 — Lista de compra
1. NÍVEL DE CONFIANÇA
2. RESUMO DE COMPRA
3. ITENS / UNIDADES / QUANTIDADES
4. MEMÓRIA DE CÁLCULO da origem dos quantitativos
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 5 — Conferência de orçamento
1. NÍVEL DE CONFIANÇA
2. RESUMO EXECUTIVO
3. ITENS CONSISTENTES E ITENS PROBLEMÁTICOS
4. MEMÓRIA DE CÁLCULO das verificações críticas
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 6 — Análise de BDI
1. NÍVEL DE CONFIANÇA
2. RESUMO DO BDI
3. COMPONENTES E FAIXAS DE REFERÊNCIA
4. MEMÓRIA DE CÁLCULO, se refeita
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 7 — Proposta de subcontratado
1. NÍVEL DE CONFIANÇA
2. RESUMO EXECUTIVO
3. ESCOPO / PREÇO / RISCOS
4. MEMÓRIA DE CÁLCULO das divergências relevantes
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Template 8 — Comparação entre bases
1. NÍVEL DE CONFIANÇA
2. RESUMO COMPARATIVO
3. DIFERENÇAS DE COMPOSIÇÃO, PREÇO E PREMISSA
4. MEMÓRIA DE CÁLCULO das diferenças relevantes
5. PONTOS DE ATENÇÃO
6. PARECER TÉCNICO

## Classificação dos pontos de atenção
Sempre classificar como:
- CRÍTICO
- ATENÇÃO
- OBSERVAÇÃO

E declarar impacto técnico, comercial, contratual ou de prazo.
