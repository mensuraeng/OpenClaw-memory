# XP - template de segredos e acessos

Preencher fora da memória textual, preferencialmente em KeePassXC.

## Campos esperados
- nome da organização
- login owner do portal developer
- client_id
- client_secret
- certificate / key, se exigido
- redirect_uri, se aplicável
- environment/base_url
- scopes liberados
- observações de aprovação/contrato
