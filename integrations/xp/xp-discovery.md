# XP Discovery

_Atualizado em 2026-04-20_

## Sinais públicos confirmados
- existe portal público: `https://developer.xpinc.com/`
- existe área de documentação: `https://developer.xpinc.com/documentacao`
- existe catálogo de APIs: `https://developer.xpinc.com/apis`
- o portal público comunica foco em parceiros com acesso a dados de:
  - captação
  - posição
  - movimentação
  - produto
  - conta
  - comissão
- parte da documentação e do catálogo aparenta exigir login para ver detalhes e subscriptions

## Sinais ainda não confirmados neste runtime
- APIs específicas para pagamentos
- APIs específicas para Pix
- APIs específicas para comprovantes bancários
- escopo exato para conta empresarial da MIA
- exigência de contrato/parceria para habilitação

## Leitura atual
A XP parece ter ecossistema de APIs real, mas não está comprovado ainda que a conta operacional da MIA tenha acesso direto ao conjunto necessário para:
- leitura bancária da conta
- iniciação de pagamento
- aprovação operacional

## Próximas validações necessárias
1. login no portal XP Developer com credencial habilitada
2. inspeção das APIs disponíveis para a organização da MIA
3. confirmação dos requisitos de autenticação
4. confirmação do escopo funcional mínimo para Fase 1
