# Checklist de implantação XP

_Atualizado em 2026-04-20_

## Acesso
- [ ] Confirmar se a organização MIA possui acesso ao XP Developer
- [ ] Confirmar qual login é o owner técnico
- [ ] Confirmar se existe app registration
- [ ] Confirmar se há necessidade de aprovação comercial/parceria

## Escopo
- [ ] Confirmar API de saldo
- [ ] Confirmar API de extrato
- [ ] Confirmar API de movimentação
- [ ] Confirmar API de comprovantes
- [ ] Confirmar API de pagamentos/Pix

## Segurança
- [ ] Guardar credenciais somente em cofre seguro
- [ ] Não salvar segredo em memória textual
- [ ] Definir ambiente de teste/controlado

## Operação
- [ ] Definir quais contas/documentos entram primeiro
- [ ] Definir regra de conciliação
- [ ] Definir regra de aprovação humana
- [ ] Definir regra de logging e auditoria
